Flappy Bird

O projeto Flappy Bird foi criado com base em aulas ministradas no curso fornecido pela Cod3r no ambiente de aprendizagem Udemy com objetivo de colocar em prática o que foi ensinado durante as aulas.

Projeto Galeria

O projeto Galeria foi criado utilizando as tecnologias Bootstrap, Webpack e Jquery, para colocar em prática o que foi aprendido no decorrer das aulas do curso da Cod3r.


Projeto Calculadora

A calculadora, baseada no software de calculadora da Apple, foi feita a partir do React JS, utilizando de javascript em toda a lógica, e Bootstrap e Css para aplicação do estilo.

Projeto Cadastro de Usuário

O projeto cadastro de usuário teve como foco colocar em prática conhecimentos em React Native, utilizando um arquivo JSON para manipulação dos usuários a serem colocados e retirados no modelo.
